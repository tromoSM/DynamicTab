# DynamicTab
#### DynamicTab is a more simple, modern, customizable and reliable new tab page for desktop web browsing. DynamicTab supports custom wallpapers, custom favicons, 4 major search engines, 13 popular sites as search engines and plenty of useful features on the way!.
>[!note]
>DynamicTab currently doesnt support webkit based browsers nor mobile browser. Webkit support will be available soon.
- [preview](https://tromosm.github.io/DynamicTab/)
